Create fixed-point numeritype object

create a 9 bit fixed-point numeritype object with 1 bit for sign, and 8 bits for fractions

nbit = 8;
T = numerictype(1,nbit+1,nbit);

Generate two input numbers

a = rand(1)
b = rand(1)

a =

     4.172069997617962e-01


b =

     8.580955347448873e-01

Addition

a + b in double precision:

c_add_d = a + b

% a + b in 8-bit fixed-point:
c_add_fi = fi_op(a, b, T, '+')

disp('error: c_add_d - c_add_fi')
c_add_d - c_add_fi.double

c_add_d =

     1.275302534506684e+00


c_add_fi = 

     9.960937500000000e-01

          DataTypeMode: Fixed-point: binary point scaling
            Signedness: Signed
            WordLength: 9
        FractionLength: 8
error: c_add_d - c_add_fi

ans =

     2.792087845066835e-01
