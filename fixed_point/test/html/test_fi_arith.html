Create fixed-point numeritype object

create a 9 bit fixed-point numeritype object with 1 bit for sign, and 8 bits for fractions

nbit = 8;
T = numerictype(1,nbit+1,nbit);

Generate two input numbers

a = 0.328

b = 0.179

% a in 8-bit fixed point:
afi8 = trun(a, T)

% b in 8-bit fixed point:
bfi8 = trun(b, T)

Addition
